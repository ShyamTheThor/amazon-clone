# 🛒 Amazon Clone (Frontend)

A responsive **Amazon homepage clone** built using **HTML and CSS**, replicating the UI and layout of the original Amazon website.

🔗 **Live Demo:** https://amazon-clone-five-cyan.vercel.app

---

## 🚀 Features

* 🧱 Amazon-like homepage layout
* 🎨 Clean UI using pure CSS
* 🖼️ Product sections and banners
* 🔍 Navigation bar with search box

---

## 🛠️ Tech Stack

* HTML5
* CSS3

---

## 📂 Project Structure

```
amazon-clone/
│── index.html
│── style.css
│── images
```

---

## ⚙️ Installation & Run Locally

```bash
git clone https://github.com/your-username/amazon-clone.git
cd amazon-clone
```

Then open `index.html` in your browser.
